# 🧩 Descripción

Este script en PHP muestra ejemplos básicos de operaciones matemáticas y el uso del orden de precedencia en expresiones aritméticas.
Es ideal para principiantes que están aprendiendo los fundamentos de PHP y cómo funcionan las operaciones matemáticas dentro del lenguaje.

# ⚙️ Requisitos
PHP instalado (versión 7 o superior recomendada)
Servidor local como:
XAMPP
WAMP
MAMP
o servidor integrado de PHP

# 📂 Estructura del código
Uso de variables ($numero1, $numero2)
Operadores aritméticos:
+ Suma
- Resta
* Multiplicación
Uso de echo para imprimir resultados en pantalla
Uso de <br> para saltos de línea en HTML

# 🎯 Objetivo educativo
Este código tiene como propósito:

Entender operaciones matemáticas básicas en PHP
Aprender cómo funciona el orden de ejecución de operaciones
Practicar el uso de variables y salida en pantalla
